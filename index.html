The Chain of Command Pattern

The controller pattern that will rock your face off!


Read Online Read Online

Chain of Command (CoCo) is a design pattern for middleware. Map an entry point (e.g. a route) to a sequence of actions (a chain of commands), each of which has access to the output of all previous commands.

CoCo works well for web applications, command-line utilities, and programs that process data sequentially. While no framework solves all problems, CoCo provides an architecturally oriented framework for creating many different kinds of applications.
